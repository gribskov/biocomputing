FastA Format

FastA format is the most common format used for online servers. A FastA file begins with > followed by the sequence ID and a text comment. The sequence ID starts after the > and continue to the first space. The comment is optional; in this case the ID ends at the end of the line. All other lines are the sequence with no spaces or numbers added.
